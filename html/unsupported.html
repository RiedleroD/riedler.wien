Uh-Oh!

Looks like you're using an unsupported browser!

This is not because your browser lacks features, but for ethical reasons - Gianmarco said it best, so here you go.
Note that this cannot be a false positive, as I check for an attribute within the javascript environment of the browser that tells me whether the browser is chromium based or not. If your browser marks itself as Chrome, that's its fault, not mine.
Note on that note: IE10 marked itself as chrome to ward off websites that tried to block non-chrome browsers (for featureset reasons) - I just hope you're not using IE10 is all

In any case, I recommend a Firefox-based browser, though I don't block Safari either. Take a look at this:

Desktop

  • Firefox - Plain and main, this is for the average Joe. It's what I use and test my website on.
  • Waterfox - Firefox, with minor improvements. I haven't tried it myself :shrug:
  • Librewolf - A privacy-centered Variant of Firefox.
  • TOR Browser - It's also based on FF. Not really meant for normal use, but I thought you might want to know that anyway.
  • GNU IceCat - A free as in freedom FF variant. I get the feeling this is not what you care about.

Mobile (mostly Android, since… yknow, sideloading)

I generally recommend using FFUpdater for easier updates of FF-based browsers outside the play store.
  • Firefox - Yes, also avaliable on mobile. Note that there's limited extension support - including uBlock Origin and ABP!
  • Iceraven - Basically just Librewolf, but on mobile. There's some more extensions enabled here.
  • Mull - Very similar to mobile Firefox, but with a few small improvements. It's what I use and test my website on.

Yes, there's still other browsers. You could use IE11. Or WebKitGTK. Or Konqueror. But those are all pretty crap, since it's near impossible to keep up with …mainly google's pace of adding new features to the web standards. Mozilla is big enough and Apple can afford to not care, but all the others get left behind :(
